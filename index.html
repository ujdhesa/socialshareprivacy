  3. Domosdoshmëri dhe Kufizime

    Domosdoshmëri teknike janë pasja e jQuery-it dhe Javascript-it të aktivizuara te shfletuesi juaj. Programin e kemi testuar me jQuery 1.4.

    Nëse keni sugjerime për përmirësime, ju lutemi, lidhuni me ne përmes 2klick@heise.de.

    Aktivizimi përgjithmonë i shërbimit funksionon vetëm me Internet Explorer versioni 8, pasi versionet e mëparshëm nuk mbulonin JSON-in. Ndaj edhe mungon kjo veçori te IE <= 7 . Pjesa tjetër e shtojcës nuk ndikohet kjo.

    Për Facebook-un nevojitet një App-ID, por tashmë jo. Shihni shënimet në lidhje me Facebook-un.

  4. Përmasa

    Shtojca lyp gjithsej rreth 600 piksela gjerësi (nëse funksionojnë krejt shërbimet) dhe rreth 290 piksela vertikalisht, por kjo sigurisht që varet nga gjatësia e tekstit të dhënë për ndihmëzën.

  </script>

    2. Shpjegimi i kodit

      <script type="text/javascript" src="jquery.js"></script> 
<script type="text/javascript" src="jquery.socialshareprivacy.js"></script>

      Rreshti i parë përfshin platformën Javascript „JQuery“ (http://jquery.com/), rreshti i dytë shtojcën tonë. jQuery nuk përfshihet në paketën tonë, duhet ta shkarkoni nga site-i i vet web. 

      <script type="text/javascript">
  jQuery(document).ready(function($){
    if($('#socialshareprivacy').length > 0){
      $('#socialshareprivacy').socialSharePrivacy(); 
    }
  });
</script>

      Në këtë <script> funksioni i shtojcës i bashkangjitet te faqja një elementi HTML të zbrazët, arbitrar, në këtë rast elementi me id socialshareprivacy. Që të sigurohet prania reale e objektit, kemi shtuar funksionin e kontrollit if përmbyllës të bashkangjitjes, që të kontrollohet për kushtet.

      <body>
  …
  <div id="socialshareprivacy"></div>
  …
</body>

      Diku te ndarja <body> e site-it web vendosët një element HTML të zbrazët me id-në e dëshiruar, dhe të përdorur për të njëjtën id që keni dhënë në <script>-in e mëparshëm.

  6. Mundësi

    Mund të kihen disa mundësi integrimi. Tek sa vijon, radhiten mundësitë e përgjithshme në fillimin dhe mandej mundësitë për shërbimet një e nga një (Facebook, Twitter, Google+).
    Shembull thirrjeje me mundësi: 

    $('#socialshareprivacy').socialSharePrivacy({
  services : {
    facebook : {
      'perma_option': 'off'
    }, 
    twitter : {
        'status' : 'off'
    },
    gplus : {
      'display_name' : 'Google Plus'
    }
  },
  'cookie_domain' : 'heise.de'
});

  7. Shembuj Integrimi

    Sa vijon janë pak shembuj integrimi sipas formësimesh të përdorura shpesh.

    • Shfaq Vetëm Facebook-un

      $('#socialshareprivacy').socialSharePrivacy({
    services : {
        twitter : {
            'status' : 'off'
        },
        gplus : {
            'status' : 'off'
        }
    }
});

    • Mos e ofro mundësinë e aktivizimit përgjithmonë

      $('#socialshareprivacy').socialSharePrivacy({
    services : {
        facebook : {
            'perma_option'  : 'off'
        }, 
        twitter : {
            'perma_option' : 'off'
        },
        gplus : {
            'perma_option' : 'off'
        }
    }
});

    • Ofro vetëm Google+ dhe jepni shtegun për te një kartelë CSS e juaja.

      $('#socialshareprivacy').socialSharePrivacy({
    services : {
        facebook : {
            'status' : 'off'
        }, 
        twitter : {
            'status' : 'off'
        }
    },
    'css_path' : '/style/plugins/socialshareprivacy.css'
});

    • Disa shtylla butonash 2-click njëherazi në një faqe

      Mundësia 1: Thirrje e shtojcës me një përzgjedhës të përshtatshëm

      <div class="something">
    <h3><a href="http://www.heise.de">heise</a></h3>
    <p>lorem ipsum</p>
    <div class="social"></div>
</div>

<div class="something">
    <h3><a href="http://www.heise.de/security/">heise security</a></h3>
    <p>dolor sit amet</p>
    <div class="social"></div>
</div>

<script>
$(".social").socialSharePrivacy({
    uri : function(context) {
        return $(context).parents(".something").find("h3 a").attr("href");
    }
});
</script>

      Mundësia 2: Thirrje të shumëfishta të shtojcës

      <div>
    <h3><a href="http://www.heise.de">heise</a></h3>
    <p>lorem ipsum</p>
    <div id="one"></div>
</div>
<script>
$("#one").socialSharePrivacy({
    uri : "http://www.heise.de"
});
</script>

<div>
    <h3><a href="http://www.heise.de/security/">heise security</a></h3>

  8. URL

    URL-ja që i jepet shërbimit mund të kontrollohet përmes një mundësie. Si parazgjedhje, përdoret një funksion brenda shtojcës, dhe URL-ja e dokumentit në faqen e atëçastshme merret prej document.location.href, por, nëse URL kanonikale është e rregulluar ( ) atëherë përdoret kjo e fundit.

  9. Rregullime Për Mbajtjen Mend

    Përpara se të kërkohet cookie për kontrollin e parapëlqimeve të përdoruesit, kontrollojmë së pari nëse shtojca është e formësuar. Nëse shtojca është ndryshuar më pas, përdoruesi nuk pëson gjë.
    Për krejt shërbimet, nëse funksioni i ruajtjes është çaktivizuar ('perma_option' : 'off') atëherë menuja e rregullimeve nuk shfaqet.
